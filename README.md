# Rahmet Labs Website

Modern, responsive website for Rahmet Labs built with React and Material-UI.

## Features

- 🎨 Modern UI with Material Design
- 🌍 Internationalization (English and Russian)
- 📱 Fully Responsive
- ⚡ Performance Optimized
- 🔍 SEO Ready
- 🎭 Smooth Animations

## Tech Stack

- React
- Material-UI
- i18next
- Framer Motion
- React Helmet Async

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/rahmet-labs/website.git
cd website
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm start
# or
yarn start
```

4. Build for production
```bash
npm run build
# or
yarn build
```

## Project Structure

```
src/
├── components/     # React components
├── locales/       # Translation files
├── styles/        # Global styles
├── App.js         # Main app component
├── i18n.js        # i18next configuration
└── theme.js       # Material-UI theme
```
